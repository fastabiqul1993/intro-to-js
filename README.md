### 01. Before run
---------------------------------------
Make sure [Node.js](https://nodejs.org/en/) already installed in your computer.


### 02. How to run
---------------------------------------
Insert each of the folder for running the task.

```

$ node arrayOfObj.js

# Task 2
$ node 01-editObject.js
$ node 02-mesinPrinter.js
$ node 03-meanMedianModus.js.js

# Task 3
```
